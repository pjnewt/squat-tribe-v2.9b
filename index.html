Info

Squat Tribe is a structured, data-driven training app designed to help you build lower-body strength using five squat variations. It guides each session automatically, tracks performance, and adapts to fatigue.

Protocol:
1. Back Squat
2. Bulgarian Split Squat
3. Front Squat
4. Side Step Squat
5. Sumo Squat

Workout structure:
Anchor set to fatigue, then repeated Myo sets using a fixed target based on 20% of Anchor reps.

Unilateral logic:
Bulgarian Squat and Side Step start with the weaker side. The second side matches reps and sets exactly. Only time differs.

Metrics:
Mechanical Load Score (MLS) = Load × Reps
Total Reps Density Score (TRDS) = MLS ÷ Time

Load is calculated as:
Bilateral = external load + (bodyweight × 0.70)
Unilateral = external load + (bodyweight × 0.85)
